The following API's are supported by this SDK.
API documentation can be found at http://myallocator.github.io/myallocator-php-sdk-ota-docs/

Inbound (MA -> OTA):

  HealthCheck
  CreateProperty
  SetupProperty
  GetRoomTypes
  GetBookingId
  GetBookingList
  GetRatePlans
  ARIUpdate

Outbound (OTA -> MA):

  NotifyBooking
  UpdateProfile
